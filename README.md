# Crop Stress Early Warning System (CSEWS)
### MSc Data Science Dissertation | 2024-25
### Focus: Vidarbha and Marathwada, Maharashtra, India

## Project Overview
An AI system that predicts crop stress 7 to 14 days before it
becomes visible to farmers, using free satellite data and
open-source machine learning tools.

## Four Modules
- Module 1: Leaf Disease Detection (EfficientNet-B4)
- Module 2: Satellite NDVI Stress Forecasting (LSTM)
- Module 3: Multilingual RAG Advisory (IndicBERT + Groq)
- Module 4: RL Intervention Optimizer (PPO + AquaCrop)

## Data Sources
- Sentinel-2 satellite data via Element84 Earth Search API (free)
- NASA POWER weather data (free)
- PlantVillage dataset (free)
- ICAR advisory documents (free)
